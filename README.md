# Portfolio Website 🌐

A modern and responsive personal portfolio website built to showcase professional work, skills, projects, and achievements.

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern UI with smooth animations
- Dark / Light theme support
- Interactive navigation
- Project showcase with category filtering
- Skills and technology sections
- Contact form integration
- Social media links
- Smooth scrolling effects
- Animated sections using AOS library

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- Font Awesome
- AOS (Animate On Scroll)
- Google Fonts

---


---

This website is deployed using:

**GitHub Pages**
